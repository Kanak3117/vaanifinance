# 💰 VaaniFinance — AI Financial Advisor for Rural India
> Hack to the Future Hackathon | Vernacular Finance Advisor


## 🌾 The Problem

Millions of people in rural and semi-urban India lack access to simple financial guidance.  
Complex language, limited access to banks, and low awareness prevent them from making informed financial decisions.
**Result:** Savings remain unused and growth opportunities are missed.


## 💡 Our Solution
VaaniFinance is a mobile-first AI financial advisor that explains money concepts in **simple local languages** — Hindi, Tamil, Bengali, Marathi, and English.
It feels like guidance from a trusted elder — not a technical chatbot.


## ✨ Features
- 🤖 AI Chat — 4-line simple financial explanations  
- 📊 Dashboard — Goal tracking with time estimation  
- 🧮 Calculator — SIP, FD, PPF, RD calculations  
- 🏛️ Schemes Finder — Govt schemes based on user profile  
- 🎯 Finance Quiz — Learn with quick questions  
- 🔊 Voice Input/Output — Speak and listen  
- 📤 WhatsApp Share — Share advice easily  
- 🔥 Streak Tracker — Build financial habits  


## 🗣️ Languages Supported
- हिन्दी (Hindi)  
- தமிழ் (Tamil)  
- বাংলা (Bengali)  
- मराठी (Marathi)  
- English  


## 🏗️ Tech Stack
- Frontend: React.js  
- AI: Groq API (LLaMA 3.3 70B)  
- Voice: Web Speech API  
- Storage: sessionStorage  
- Deployment: Vercel
